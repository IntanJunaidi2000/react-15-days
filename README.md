# ⚛️ 15-Day React Challenge

One React project a day for 15 days — built to sharpen my frontend skills and grow my portfolio as a full-stack developer.

**Goal:** Go from React basics to 15 deployable projects, covering `useState`, APIs, full-stack integration with Spring Boot, and finishing with a complete portfolio site on Day 15.

## Tech Stack

- React (Vite)
- JavaScript
- CSS (no frameworks — vanilla all the way)
- External APIs (MealDB, GitHub, OpenWeatherMap, OMDB)
- Spring Boot backend integration (Day 14)

## Project Structure

```
react-15-days/
├── day-01-profile-card/
├── day-02-mood-tracker/
├── day-03-pomodoro-timer/
└── ...
```

Each folder is a standalone Vite + React project. To run any project:

```bash
cd day-01-profile-card
npm install
npm run dev
```
